# Weibo-plugin
- 批量删除微博脚本

# 安装说明
- 因为Chrome商店开发者注册收费，穷逼学生党，所以没有上架
- 所以按照以下操作安装：
<!-- - 所以提供以下2中方式使用： -->
<!-- > **方案.1：** -->
> - clone本仓库到本地
> - 在Chrome扩展管理页打开开发这模式
> - 点击加载已解压的扩展程序

# 使用说明
- 登陆网页版微博
- 切换到新版UI
- 进入个人主页，在筛选中过滤查出想删除的微博
- 然后点击顶部导航栏头像后的删除按钮即可
- 删除过程中全程自动，可以按键盘的ESC停止删除
